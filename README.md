# Aashish Shrestha — Portfolio

A clean, lightweight portfolio website built with vanilla HTML, CSS, and JavaScript.

## Files

- **index.html** - Main HTML structure
- **styles.css** - All styling (no frameworks)
- **script.js** - Interactivity (expandable sections, animations)

## Running the Site

Simply open `index.html` in a browser, or use a local HTTP server:

```bash
# Python 3
python3 -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server

# PHP
php -S localhost:3000
```

Then open `http://localhost:8080` (or your chosen port) in your browser.

## Features

- Fully responsive design
- Expandable case studies with technical deep-dives
- Scroll-triggered animations
- Contact form
- Zero dependencies
- No build process required

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
